# To-Do List (Java Console App)

## Project Description
This is a simple **menu-driven To-Do List application** built using **Java**. The program runs in the console and allows a user to manage tasks through a text-based menu.

This project was created to practice **Java fundamentals** such as loops, conditionals, lists, user input, and basic program structure.

## Features
- Add new tasks
- View all tasks
- Remove tasks by number
- Mark tasks as done
- Menu-based navigation
- Input validation
- Clean exit option
